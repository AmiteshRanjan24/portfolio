## How to Run Locally

```bash
# Clone the repository
git clone https://github.com/AmiteshRanjan24/portfolio.git

# Navigate to project folder
cd my-portfolio

# Install dependencies
npm install

# Start development server
npm run dev

Open http://localhost:5173 within your browser to see the result.
```
